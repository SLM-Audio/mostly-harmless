Detailed Description

Convenience wrapper layer implementing the creation and management of a choc::ui::WebView.

Provides the basic setup code for the Webview, but you'll likely want to derive from this, and override some of its functions. The underlying choc::ui::WebView can be accessed via the protected m_internalWebview member.
