Detailed Description

template<marvin::FloatType SampleType>
class mostly_harmless::Plugin< SampleType >

The base class for Plugins to subclass.

To register your subclassed plugin type with the clap factory, you must call MOSTLYHARMLESS_REGISTER(YourPluginType) at the end of your plugin source file.

To establish parameters with the plugin, create a vector of them, and std::move them into the params arg here. (In my opinion) the cleanest way to do so is with a free function - for example:

std::vector<mostly_harmless::Parameter<float>> createParams() noexcept {
return {
mostly_harmless::Parameter<float>{ 0, "MyParam", "Plugin", { 0.0f, 1.0f }, 1.0f, CLAP_PARAM_IS_AUTOMATABLE }
}
}
class YourPluginType : public mostly_harmless::Plugin<float> {
public:
YourPluginType(const clap_host* host) : mostly_harmless::Plugin<float>(host, createParams()) {
}
// The rest of the boilerplate
};
